12. Unit Testing¶

This section is mainly for those who wish to make tweaks to AIMBAT themselves. We have added some unit tests to AIMBAT to ensure that it is robust. See the Python Unit Testing Framework for more details.

12.1. Running the Tests¶

In the AIMBAT repository, cd into /src/pysmo/unit_tests and run:

python run_unit_tests.py
